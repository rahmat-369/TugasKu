Tentang TugasKu

Aplikasi Pengelola Tugas dan Jadwal

TugasKu membantu kamu mengatur tugas sekolah dan jadwal pelajaran dengan mudah.

Fitur Utama

  • 📝 Tambah dan kelola tugas harian
  • 📅 Jadwal pelajaran mingguan
  • 📝 Catatan pribadi
  • 🏷️ Filter tugas berdasarkan status dan deadline
  • 🎨 Pilihan tema yang beragam
  • 💾 Data tersimpan di browser (localStorage)
  • 📱 Responsif untuk mobile dan desktop
  • 🤖 AI Assistant untuk input otomatis
